# Fopusha 水墨介绍片设计

## 目标

制作一支约 3 分钟的中文介绍视频，整体风格参考中国水墨动画。视频需要让首次接触 `Fopusha` 的人知道：

- 这个网站是做什么的
- 可以完成哪些礼佛与祭祖相关操作
- 如何大致使用
- 网站的气质和特色是什么

## 时长分配

- 第一段：40 秒
- 第二段：85 秒
- 第三段：55 秒

总时长约 180 秒。

## 风格

- 视觉：宣纸、水墨晕染、金色佛光、卷轴式 UI 浮现
- 转场：墨色融开、收束、再晕染展开
- 音乐：柔和古琴、箫、轻氛围
- 配音：温柔中文女声

## 结构

### 第一段：这一方清净之境

回答“Fopusha 是什么”。先建立情绪与整体认知，不急于解释所有功能。画面以佛像、莲灯、香烟和首页 3 个主入口构成。

### 第二段：循着香火，完成一念心愿

回答“怎么使用”。重点展示礼佛流程，同时用较短篇幅带出祭祖流程。这里是整支片最长的部分。

### 第三段：愿有所归，心有所安

回答“这个网站为什么值得记住”。强调网站的安静气质、清晰流程、说明与引导、礼佛与祭祖并行的特点，并以品牌落点收束。
